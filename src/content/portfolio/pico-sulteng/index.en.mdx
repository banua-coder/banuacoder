## Context

PICO Sulteng (Platform Informasi COVID-19 Sulawesi Tengah) was built in 2021 when the
COVID-19 pandemic was still in its active phase. This was a high-urgency project — regional
government needed a platform that could go live in weeks, not months.

The project taught Banua Coder a great deal about fast delivery under pressure: from
prioritising scope and keeping infrastructure stable under traffic spikes, to coordinating
with government stakeholders who have different approval processes from commercial clients.

## What We Built

PICO Sulteng consists of three components:

**1. REST API (Laravel Backend)**
- Case statistics endpoints that operators can update
- Healthcare facility data management
- Public endpoints consumed by both the mobile app and web portal

**2. Mobile Application (Flutter)**
- Case statistics dashboard by province and district/city
- Interactive case distribution map
- COVID referral hospital and health centre directory
- Health protocol and quarantine guidelines

**3. Web Portal**
- Web version accessible from a browser without downloading an app
- Shares the same data as the mobile app via the API

## Impact

PICO Sulteng launched successfully and became the official COVID-19 information platform
for Central Sulawesi Province. The platform was archived after the pandemic subsided — a
project that accomplished its mission within its context.

For Banua Coder, this project is a reference for the capability to build multi-layer public
systems with fast delivery in emergency conditions.
