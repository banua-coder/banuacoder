## Context

Patonro is the regional tax and levy services application for Gowa Regency, South Sulawesi,
developed in partnership with PT Brantas Inti Utama as part of a public service modernisation
effort. In the public sector context, reliability and data security standards are higher than
typical commercial applications — the system must be trusted by residents entrusting their
regional payment obligations to it.

This project gave Banua Coder deep understanding of public-sector service workflows:
from complete audit trail requirements and integration with existing government systems,
to UX that must be usable by a wide range of age groups and digital literacy levels.

## What We Built

Patonro was built with Flutter (mobile) and Laravel (backend API), with an architecture
that scales as the number of registered taxpayers grows:

- **Bill dashboard**: A summary view of all active taxpayer bills, grouped by tax/levy type.
- **Payment history**: Verified transaction log with status and downloadable payment receipts.
- **Direct payment**: Payment gateway integration allowing bill payment without leaving the app.
- **Due-date notifications**: Push notifications 7 days and 1 day before the deadline to
  reduce arrears.
- **Tax object search**: Search by tax object number for taxpayers with multiple assets.

## Impact

Patonro is deployed and actively used by regional taxpayers. The most recent update in
December 2025 confirms the application is in an active development cycle.

This project reinforces Banua Coder's position as a team capable of handling public-sector
systems with stricter security and reliability standards.
