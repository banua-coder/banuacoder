## Context

Reab is a social/media application developed for Ulearna Technology LTD, a technology company
operating in international markets. This was not an internal MVP — it is a live product published
to a broad user base with consumer-grade quality expectations.

Banua Coder's involvement reflects the client's confidence in the team's ability to meet
engineering standards above those typical of local projects: from architecture and performance
to the end-user experience.

## What We Built

We worked on Reab's mobile application development using Flutter, ensuring a clean and
well-structured codebase that makes ongoing feature development manageable. Our primary focus areas included:

- **Feed performance**: Optimizing long content list rendering so scrolling stays smooth
  even on mid-range devices.
- **Scalable architecture**: Clear separation between UI, domain logic, and data layers
  so new features can be added without restructuring the existing codebase.
- **Backend integration**: Robust REST API consumption with error handling and caching
  strategies to minimize perceived latency.
- **Push notifications**: Firebase Cloud Messaging integration for real-time notifications.

## Impact

Reab has been successfully published to Google Play and downloaded over 50,000 times.
The application continues to be actively maintained, with the most recent update in February 2026 —
evidence that the system is solid enough for long-term engagement.

This project is one of Banua Coder's primary references for demonstrating capability in
handling consumer-facing products in an international production environment.
