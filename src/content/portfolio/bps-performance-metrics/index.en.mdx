## Context

BPS Performance Metrics is an internal dashboard platform built to support staff performance
monitoring at Badan Pusat Statistik, Indonesia's national statistics agency responsible for
collecting, processing, and disseminating official statistical data.

Working with an institution of BPS's scale means dealing with significant data complexity,
strict security requirements, and users with varying technical backgrounds. This project
demonstrates Banua Coder's capability in delivering dashboard and data visualization work
for national-scale government agencies.

## What We Built

The dashboard platform was built as an internal web application with a focus on data clarity
and ease of use:

- **Key performance indicators (KPIs)**: Individual and team KPI displays in easily readable
  visual formats — gauges, progress bars, and trend lines for periodic monitoring.
- **Cross-unit comparison**: Ability to compare performance across work units to identify areas
  needing attention or worth using as benchmarks.
- **Target tracking**: Visualization of quarterly and annual target achievement with alert
  indicators for targets at risk of not being met.
- **Report export**: Data export functionality in formats supporting formal reporting and
  internal audit needs.

## Impact

The BPS Performance Metrics dashboard was successfully implemented and is actively used as
an internal performance monitoring tool. This project strengthens Banua Coder's portfolio
in data-driven tools for national government institutions — with BPS as one of the most
prestigious public sector client references.
